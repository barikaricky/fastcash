# Infrastructure Configuration

This directory contains infrastructure-as-code and deployment configurations for FastCash.

## Structure

- **docker/** - Docker Compose and container configurations
- **k8s/** - Kubernetes manifests and Helm charts
- **terraform/** - AWS infrastructure provisioning
- **ci-cd/** - GitHub Actions workflows and deployment pipelines
- **monitoring/** - Grafana, Prometheus, and logging configurations

## Usage

See individual subdirectories for specific configuration details.
